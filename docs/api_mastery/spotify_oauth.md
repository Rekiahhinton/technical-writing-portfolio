## Overview

OAuth 2.0 is a universal authoriztion framework. The high-level security measures allow developers to access sensitive user data with proper authentication. 

This guide teaches REST-familiar developers how to authenticate with the Spotify Web API using OAuth 2.0 to gain access to Spotify's music catalog and user information. 

Specifically, this guide will touch on two endpoints: 

1. ``https://accounts.spotify.com/authorize`` - Gaining Access to User Information

2. ``https://accounts.spotify.com/api/token`` - Exchange Authorization Code for Tokens

By the end, developers will be able to build apps with new functionality with access to the Spotify Web API.
