### Introduction
Meta prompting is a useful tactic for extrapolating specific responses from a language model (LM) by asking the model to create a prompt which would yield the desired response. By requesting the language model (LM) to create a prompt to yield a particular result, the skilled technical writer can "reverse engineer" documentation by using samples of documentation that have already proven successful.
