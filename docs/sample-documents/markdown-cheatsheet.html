Inline Images

Like a link, but starts with !
You type
![description](image-url.png)

![alt text](photo.jpg "Hover text")
1
Start with ! — this tells Markdown it's an image
2
[description] — text shown if image fails to load
3
(url) — the image address. Add "text" inside for hover tooltip
You get
🖼
← image renders here
If the image can't load, the description text appears instead.
Key insight: A regular link is [text](url). An image is just ![text](url) — same pattern, add the !.

Code Blocks

Wrap any code in triple backticks
You type
```
your code goes here
multiple lines work too
```

With syntax highlighting:
``` python
def greet(name):
    print("Hello, " + name)
```
Tip: The 3 backticks must be on their own line, above and below your code.
You get
your code goes here
multiple lines work too
WITH PYTHON HIGHLIGHTING:
def greet(name):
    print("Hello, " + name)
Supported languages include: python, javascript, go, sql, bash, java, css, and 15+ more.

Tables

The most visual gap between syntax and result
You type
| Name    | Score | Grade |
| :------ | :---: | ----: |
| Alice   |  95   |   A   |
| Bob     |  82   |   B   |
| Carol   |  74   |   C   |
1
Row 1 — your column headers, wrapped in | pipes
2
Row 2 — dashes --- under each header. This is required.
3
Row 3+ — your data, same pipe structure
Alignment with colons:
:--- = left  |  :---: = center  |  ---: = right
You get
Name Score Grade
Alice95A
Bob82B
Carol74C
Don't panic: The columns don't need to line up perfectly in your raw text — Markdown still renders the table correctly.

Nested Lists

Indentation is everything
You type
* Fruits
  * Citrus
    * Lemon
    * Lime
  * Berries
* Vegetables
  * Root vegetables
1
Top level: * or - at the start of the line
2
Sub-item: add 2 spaces before the bullet
3
Sub-sub-item: add 4 spaces (2 more per level)
Common mistake: Using 1 space or a tab instead of exactly 2 spaces — this breaks the nesting.
You get
  • Fruits
    • Citrus
      • Lemon
      • Lime
    • Berries
  • Vegetables
    • Root vegetables
Rule of thumb: Each level deeper = 2 more spaces at the front of the line.

Linked Images

A clickable image — combining two patterns in one
You type
[![alt text](image.png)](https://example.com)
1
Write a normal image: ![alt](image.png)
2
Wrap the whole thing in a link: […](url)
3
The !
4
The outer (url) is where the click takes you
Memory trick: Think of it as putting an image inside a link.
Link = [content](url) → make the content an image.
You get
🖼
Clicking this image opens a link →
Pattern breakdown:

[![alt text](image.png)](destination-url)

RED = image + link brackets  |  GREEN = alt text  |  PURPLE = image file  |  GOLD = link URL
